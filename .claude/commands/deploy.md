# Deploy to Cloudflare Pages

執行以下步驟將專案部署到 Cloudflare Pages：

1. 在專案目錄 `/Users/ching-wu/Desktop/claude-inspiration/trustguard` 執行 `npm run build`，確認 build 成功
2. 執行 `git add -A`
3. 執行 `git commit -m "deploy: update site"`（如果有新變更）
4. 執行 `git push`
5. 告知使用者去 `github.com/Toriwuuu/trustguard/actions` 查看部署進度，網址是 `https://trustguard-5vg.pages.dev`
